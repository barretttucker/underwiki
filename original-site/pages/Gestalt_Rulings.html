Gestalt Rulings

Gestalt Core Classes fixed for life of character: Nov. 13, 2005

The core classes you choose at level 1 to form your new gestalt class are FIXED for the life of your character. You can always choose prestige classes you qualify for at any given level and replace the related core class with the prestige class.

Normal rules: the srd says that at each new level you 'choose two classes' and combine them. Theoretically you could be a fighter rogue at level 1, a thief psion at level 2, etc. This is not how we are going to use the gestalt rules.

Prestige Class replaces related core class for gestalts: Nov. 13, 2005

When you choose prestige class(es) for leveling, replace the RELATED core class with the prestige class. For example, a thief-cleric choosing the assassin prestige class would advance as a cleric-assassin, not as a thief-assassin. If there is doubt about which is the more related, consult the DM.

Spellcasting / Manifesting Levels don't exceed +1 / level: Nov. 13, 2005

If you are able to take a prestige class that allows you to advance "+1 level of existing spellcasting class" (or manifesting class, for psions), remember that bonuses DO NOT STACK with otherwise identical abilities. For example, if you are a psion-sorceror and take the Cerebremancer prestige class (which allows you to advance as a psion and as a spellcaster), you would have to pick whether you wish to be a psion-cerebremancer or a sorceror-cerebremancer and you would only gain 1 level of manifesting/spellcasting with each character level. You would not gain +2 levels in either one at a single character level. This is a known exploit.

Note that this makes a few prestige classes irrelevant (cerebremancer among them), but that's a side effect of the power and flexibility that gestalts have over standard characters.
