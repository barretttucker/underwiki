character classes

Forbidden Classes:

  • Cleric

  • Wizard

Note that outdoor-focused powers, abilities etc have Underdark equivalents. You could easily have a Ranger, for example, skilled at traversing the wilds of the underground and being knowledgeable in the ways of fungi and mosses instead of trees.

Note that Druids and Rangers would both require a bit of work, but we can create some underground version of each if a player really really wants to.
