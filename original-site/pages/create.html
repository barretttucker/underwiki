create

To create a new page, simply type in the name of the page title desired surrounded as follows: 

["NewPage"]

The brackets and quotation marks are indicators for the wiki engine to create a new page link.

Save the page's changes and view it again. You'll see a new, grey link with your page's name.

Click on it.

It will take you to a new page and ask if you want to create a new, empty page.

Click on that link and it will open up a new page's text box for your content.
